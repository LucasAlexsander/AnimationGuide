Tag transform

Com esta propriedade conseguimos fazer algumas alterações nos elementos de forma muito interessante. Conseguimos aumentar o tamanho, arrastar para o lado e outras coisas.

Translate( )

Faz com que o elemento ande para um lado. Podemos usar como parametro (X, Y)

Temos também:

  • translateX(px, vh, ...)
  • translateY(px, vh, ...)

Scale( )

Aumenta ou diminui o tamanho do elemento, como paramentro default temos 1. Podemos usar como parametro (X, Y)

Temos também:

  • scaleX(px, vh, ...)
  • scaleY(px, vh, ...)

Rotate( )

Com ele podemos rotacionar o elemento. Podemos usar como parametro (X, Y), esses valores são em deg ou turn.

Temos também:

  • rotateX( )
  • rotateY( )
  • rotateZ( )
  • rotate3d( )

Skew( )

Dá uma inclinada no elemento. Podemos usar como parametro (X, Y), esses valores são em deg ou turn.

Temos também:

  • skewX(deg, turn)
  • skewY(deg, turn)

Transition

Essa propriedade é muito importante para definirmos como será feita a transição no elemento, se será rápida ou mais lenta, se vai ter um delay ou não, e outros efeitos legais.

ShortHand: transition: ‹propriedade› ‹tempo de duração› ‹suavização› ‹delay›

transition-propety

Neste momento vamos passar as propriedades do css que irá passar pela transição.

Por exemplo:

  • transition-property: hight;
  • transition-property: opacity;

transition-duration( )

Escolhemos a duraçção que a transição vai ter.

Por exemplo:

  • transition-duration: .5s
  • transition-duration: 500ms

transition-delay

Definimos um tempo para acontecer a transição depois de uma determinada ação.

Por exemplo:

  • transition-delay: 1s
  • transition-delay: 1000ms

transition-timming-function

Definimos a suaveidade que a transição vai acontecer.

Temos também:

  • ease-in
  • ease-out
  • ease-in-out
  • cubic-bezier()

keyframe

Essa função nos ajuda a criar animações, onde vamos passar ações que vão ocorrer do começo até o fim (Sempre que possível utilize transform ou opacity).

@keyframe {
                        0% ou from {
                            // code
        }
                        100% ou to {
                            // code
        }
}

Animation

Essa propriedade é necessária quando queremos chamar o keyframe, nela vamos passar alguns paramêtros importantes.

animation-name

Vamos passar o nome da animação, onde vai estar no keyfram.

Por exemplo:

  • animation-name: rotate;
  • animation-name: transparent;

animation-duration( )

Escolhemos a duraçção que a animação vai ter.

Por exemplo:

  • animation-duration: .5s
  • animation-duration: 500ms

animation-timming-function

Como a animação vai progredir durante a sua execução.

Por exemplo:

  • ease
  • cubic-bezier
  • steps( 'number' )
  • linear

animation-delay

Definimos o delay que a animação vai ter.

Temos também:

  • animation-delay: .4s
  • animation-delay: 400ms

animation-interation-count

Número de vezes que a animação vai acontecer.

Por exemplo:

  • 2
  • Infinite

animation-fill-mode

Defini quais valoressão aplicados antes e/ou deposi da animação.

Temos também:

Forwards: Último estado da animação deve ser mantido

Backwards: Pega o estado inicial da animação e se mantém.

Both: Ambos os casos

animation-play-state

Definimos se animação esta sendo executada
